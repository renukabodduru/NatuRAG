# NatuRAG
This project is a RAG based chatbot which efficiently answers for your questions on what you have provided to the chatbot.This helps many enterprises and for people who need to what the information from the information provided by them ,not more or less by that .In Simple this bot is a  personal document Assistant.


This project is deployed in the streamlit cloud ,so you can access this from your mobile or destkop using link https://naturag-ksn9e8fakqusknlc76qrr9.streamlit.app/
